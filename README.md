# Prodigy InfoTech Web Development Internship - Task 04

## 📌 Project Overview
This repository contains the completion of **Task 4** for the **Web Development Internship** at **Prodigy InfoTech**. The objective was to design and develop a **Personal Portfolio Website** that showcases my skills, projects, achievements, education, and professional background. The portfolio is fully responsive, visually appealing, and provides visitors with an overview of my work as a web developer.

## 🛠️ Technologies Used
* **HTML5** (Structure)
* **CSS3** (Styling & Responsive Design)
* **JavaScript (ES6)** (Interactivity)

## ✨ Features Implemented
* Attractive and modern landing page.
* Responsive design for desktop, tablet, and mobile devices.
* Professional **Home** section with a profile picture and introduction.
* **About Me** section highlighting education, background, and experience.
* **Skills** section showcasing technical expertise.
* **Projects** section displaying completed work.
* **Achievements** and certifications section.
* **Contact** section with social media and communication links.
* Smooth scrolling navigation and interactive UI elements.

## 📊 Project Functionality
1. **Home Section:** Displays a professional introduction along with my profile picture.
2. **About Me:** Provides information about my education, background, and career goals.
3. **Skills:** Highlights programming languages, frameworks, and tools.
4. **Projects:** Showcases my web development projects with descriptions.
5. **Contact:** Allows visitors to connect through email and social platforms.
6. **Responsive Layout:** Optimized for desktop, tablet, and mobile devices.

## 📁 Files in this Repository
* `index.html` – Structure of the portfolio website.
* `style.css` – Styling and responsive layout.
* `script.js` – Interactive features and animations.
* `profile.jpg` *(or `profile.png`)* – Professional profile picture used in the Home section.
* `README.md` – Project documentation.
